SQL Viewer from Motion Mind

01 How to Load a Database?

Go to the home page and click on the 'Open a Database File' button. Select any .db or .sqlite file from your device to load the data instantly.

02 Navigating Tables

Once the file is loaded, you will see a list of all tables in the Sidebar on the left. Simply click on a table name to view its data grid on the right.

03 Pagination & Navigation

For large datasets, use the Prev/Next buttons at the top or bottom of the grid. To jump to a specific page, type the page number in the input box and press Enter.

04 Managing Notes

Use the 'Make a note' section from the menu to write down your observations. Notes are auto-saved locally and can be downloaded as a .txt file using the download button.
